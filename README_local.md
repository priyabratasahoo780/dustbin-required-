# Extension Archive (Dustbin)

This repository serves as an archive for all the projects and code located in the `Extension` folder. It acts as a "dustbin" or backup for various experimental, ongoing, and completed projects.

## Contents
This repository includes a wide variety of web development, C++, React, and other projects.

> **Note**: For projects that had their own Git history, their `.git` directories have been renamed to `.git_bak` to ensure all files are committed to this main repository.
