## ⚡ Performance Architecture

<div align="center">

```
╔══════════════════════════════════════════════════════════════╗
║              SMART 4-PRIORITY LOADING STRATEGY               ║
╠══════════════════════════════════════════════════════════════╣
║  ⚡ Priority 1 — Instant (0ms)                               ║
║     → Hero · Navbar · Cursor · ScrollProgress               ║
╠══════════════════════════════════════════════════════════════╣
║  🧠 Priority 2 — Crucial Info (lazy)                         ║
║     → About · Skills                                        ║
╠══════════════════════════════════════════════════════════════╣
║  🚀 Priority 3 — Heavy Interactive (lazy)                    ║
║     → Journey · Projects · LeetCode · Hackathons            ║
╠══════════════════════════════════════════════════════════════╣
║  🌐 Priority 4 — External / Footer (lazy)                    ║
║     → YouTube · Certificates · Contact · Footer             ║
╚══════════════════════════════════════════════════════════════╝
```

</div>

| Optimization | Implementation |
|---|---|
| **Code Splitting** | `React.lazy()` on all non-critical sections |
| **Memoization** | `React.memo` on every rendered component |
| **Smooth Scroll** | Lenis with exponential easing `1.001 - 2^(-10t)` |
| **Scroll Spy** | `IntersectionObserver` with `rootMargin: -30% 0px -65% 0px` |
| **Dynamic SEO** | Per-section `<title>` + meta tags update on scroll |
| **Responsive Animations** | `gsap.matchMedia()` for mobile/desktop-targeted animations |

## 🚀 Getting Started

### Prerequisites
```
Node.js  ≥ v18.0.0
npm      ≥ v9.0.0
Git
```

### 🔽 Installation

```bash
# Step 1: Clone the repository
git clone https://github.com/priyabratasahoo780/Portfol_website.git

# Step 2: Move into the project folder
cd Portfol_website

# Step 3: Install all dependencies
npm install

# Step 4: Set up environment variables
cp .env.example .env
# ✏️  Open .env and fill in your EmailJS credentials

# Step 5: Fire up the dev server
npm run dev
```

🎉 App running at: **`http://localhost:5173`**

### 📦 Build for Production

```bash
npm run build     # Create optimized production build
npm run preview   # Preview the production build locally
```

## 🔧 Environment Variables

Create a `.env` file in the root:

```env
# ─── EmailJS Configuration ────────────────────────────
VITE_EMAILJS_SERVICE_ID=your_service_id
VITE_EMAILJS_TEMPLATE_ID=your_template_id
VITE_EMAILJS_PUBLIC_KEY=your_public_key
```

> 📖 See [`EMAILJS_SETUP.md`](./EMAILJS_SETUP.md) for a step-by-step configuration guide.
