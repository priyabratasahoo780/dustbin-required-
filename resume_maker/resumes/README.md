# 💎 ResumeForge Ultra-Premium

> **Elevate your professional identity with a SaaS-level resume building experience.**

ResumeForge is a high-end, strictly light-mode resume generator designed for modern professionals. Inspired by the aesthetics of Stripe, Canva, and Linear, it offers a seamless, distraction-free environment to craft high-impact resumes with live A4 previews.

![ResumeForge Preview](https://github.com/priyabratasahoo780/Resume_generater_maker/raw/main/preview.png)

## ✨ Key Features

- **🎨 Multi-Style Architecture:** Toggle instantly between **Modern**, **Strict**, and **Creative** design directions with fluid sliding animations.
- **🚀 Full-Page Cinematic Preview:** Expand your resume to a high-fidelity, distraction-free full-screen mode for final proofing.
- **📊 AI Strength Meter:** Real-time feedback on your resume's impact and completion level.
- **🖼️ 11+ Premium Templates:** Curated layouts including the exclusive `SimpleTemplateOne`, optimized for ATS and professional aesthetics.
- **🌓 Pure Light Mode:** A deliberate design choice for a clean, professional, and high-end SaaS feel.
- **⚡ Real-time Synchronization:** See your changes instantly on a pixel-perfect A4 digital sheet.
- **📥 One-Click Export:** High-quality PDF generation with optimized asset rendering.

## 🛠️ Tech Stack

- **Core:** [React 19](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Styling:** [Tailwind CSS 4](https://tailwindcss.com/) (Next-gen CSS engine)
- **Animations:** [Framer Motion](https://www.framer.com/motion/) (Cinematic transitions & micro-interactions)
- **Icons:** [Lucide React](https://lucide.dev/) (Refined stroke-based iconography)
- **PDF Engine:** [html2canvas](https://html2canvas.hertzen.com/) & [jsPDF](https://rawgit.com/MrRio/jsPDF/master/docs/index.html)

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/priyabratasahoo780/Resume_generater_maker.git
   cd Resume_generater_maker
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Open your browser:**
   Navigate to `http://localhost:5173` to start building!
